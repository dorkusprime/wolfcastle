# Extend PromptRepository with template resolution method

Add a ResolveTemplate(name string, ctx any) (string, error) method to PromptRepository that resolves templates from the templates/ prefix (mirroring how Resolve uses prompts/). The name is relative to templates/ and uses .tmpl extension. If ctx is non-nil, parse and execute as text/template. This reuses the existing tierfs.Resolver and three-tier resolution, not a parallel system. The method should wrap errors consistently with the existing Resolve method's pattern.
