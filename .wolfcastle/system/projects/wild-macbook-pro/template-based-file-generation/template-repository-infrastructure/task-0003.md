# Define typed context structs for all template variables

Create typed structs for each template's variables in a dedicated file (e.g., internal/pipeline/template_data.go or similar). Structs needed per the spec inventory: ADRData{Title, Date, Body string}, SpecData{Title, Body string}, TaskData{ID, Title, Description, Body, Type, Class string; Deliverables, Constraints, References, AcceptanceCriteria []string}, AuditTaskData{NodeName string}. Scaffold templates (gitignore, READMEs) are static and need no struct. These structs keep template variables discoverable and type-safe.
