# Create artifact template files

Create the following template files under internal/project/templates/artifacts/: adr.md.tmpl (with {{.Title}}, {{.Date}}, {{.Body}} variables, reproducing the strings.Builder output from cmd/adr_create.go:64-75), spec.md.tmpl (with {{.Title}}, {{.Body}} variables, reproducing the fmt.Sprintf patterns from cmd/spec.go:81-86), audit-task.md.tmpl (move existing templates/audits/audit-task.md content, add {{.NodeName}} if needed), task.md.tmpl (with all TaskData fields, reproducing the strings.Builder output from cmd/task/add.go:192-200). Each template must produce byte-for-byte identical output when rendered with the same inputs as the current code.
