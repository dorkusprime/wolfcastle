# Write MigrationService contract spec

Create a contract spec for MigrationService at .wolfcastle/docs/specs/. The contract should define:

MigrationService struct with fields:
- config *config.ConfigRepository
- root string (path to .wolfcastle/)

Methods:
- MigrateDirectoryLayout() error: moves pre-ADR-077 flat layout into system/. If system/ already exists, returns nil (idempotent). Moves base/, custom/, local/, projects/, logs/ into system/. Moves loose daemon files (wolfcastle.pid, stop, daemon.log, daemon.meta.json) into system/.
- MigrateOldConfig() error: migrates old config files to three-tier layout. Moves root config.json to system/custom/config.json (only if target doesn't exist). Merges config.local.json contents into system/local/config.json (preserving existing keys).

Both methods are idempotent and non-fatal (callers log errors but don't fail).
