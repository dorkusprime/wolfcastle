# Implement ScaffoldService

Implement ScaffoldService in internal/project/scaffold_service.go. This replaces the existing standalone Scaffold(), ReScaffold(), and WriteBasePrompts() functions.

Constructor: NewScaffoldService(config *config.ConfigRepository, prompts *pipeline.PromptRepository, daemon *daemon.DaemonRepository, root string)

Init(identity *config.Identity) error:
1. Create directory tree (use os.MkdirAll for each path)
2. Write .gitignore with the whitelist pattern from existing scaffold code
3. config.WriteBase(config.Defaults()) for base config
4. config.WriteCustom(map[string]any{}) for empty custom
5. config.WriteLocal(map[string]any{'user': identity.User, 'machine': identity.Machine}) for local identity
6. Create namespace project directory at system/projects/<identity.Namespace>/
7. Write empty root index state.json
8. prompts.WriteAllBase(embedded templates) for base prompts

Reinit() error:
1. Create and call MigrationService for both migrations (log errors, don't fail)
2. Remove system/base/ entirely
3. Recreate system/base/ directory tree
4. Regenerate base config and prompts
5. Ensure custom config exists (create empty if missing)
6. Refresh identity in local config

Reference the existing scaffold.go for the exact directory paths, .gitignore content, and embedded template handling.
