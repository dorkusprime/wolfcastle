# Write ScaffoldService contract spec

Create a contract spec for ScaffoldService at .wolfcastle/docs/specs/. The contract should define:

ScaffoldService struct with fields:
- config *config.ConfigRepository
- prompts *pipeline.PromptRepository
- daemon *daemon.DaemonRepository
- root string (path to .wolfcastle/)

Methods:
- Init(identity *config.Identity) error: creates the full .wolfcastle/ structure for 'wolfcastle init'. Creates directory tree (system/base/prompts, system/base/rules, system/base/audits, system/custom, system/local, docs/, artifacts/, archive/, system/logs/). Writes .gitignore. Writes base config from Defaults(). Writes empty custom config. Writes local config with identity. Creates namespace project directory. Writes empty root index. Extracts embedded prompts via PromptRepository.WriteAllBase().
- Reinit() error: called by 'wolfcastle update'. Runs MigrationService migrations (non-fatal). Removes and recreates system/base/. Regenerates base config. Ensures custom config exists. Refreshes identity in local config.

ScaffoldService replaces the existing Scaffold(), ReScaffold(), and WriteBasePrompts() functions.
