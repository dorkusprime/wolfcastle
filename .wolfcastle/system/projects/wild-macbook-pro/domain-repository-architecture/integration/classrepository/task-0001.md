# Write ClassRepository contract spec

Create a contract spec for ClassRepository at .wolfcastle/docs/specs/. The contract should define:

- NewClassRepository(prompts *PromptRepository) constructor
- Reload(classes map[string]ClassDef) for goroutine-safe class list updates
- Resolve(key string) (string, error) with hierarchical fallback: 'lang-go' tries lang-go.md then lang.md; 'typescript/react' tries typescript/react.md then typescript.md
- List() []string returning sorted class keys
- Validate() []string returning class keys whose prompts are missing from all tiers

ClassRepository resolves class prompts through PromptRepository.ResolveRaw('prompts/classes', key+'.md'). It holds a sync.RWMutex for concurrent access: Reload takes write lock, Resolve/List/Validate take read lock.

Follow the contract spec format established by existing specs in .wolfcastle/docs/specs/.
