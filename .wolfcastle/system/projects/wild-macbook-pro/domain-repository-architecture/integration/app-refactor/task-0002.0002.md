# Migrate audit/ complex commands (approve, codebase, history, pending, reject, show)

Migrate 6 audit commands that use Resolver paths and WolfcastleDir in addition to Store mutations.

Files: audit/approve.go, audit/codebase.go, audit/history.go, audit/pending.go, audit/reject.go, audit/show.go

Patterns:
- app.RequireResolver() → app.RequireIdentity()
- app.Resolver.ProjectsDir() → app.State.Dir()
- app.Resolver.LoadRootIndex() → app.State.ReadIndex()
- app.WolfcastleDir for audit-state.json, audit-review-history.json → filepath.Join(app.Config.Root(), ...) or similar
- app.Cfg.Models[...] → cfg, _ := app.Config.Load(); cfg.Models[...]
- app.Cfg.Audit.Model → cfg.Audit.Model
- app.Cfg.Daemon.InvocationTimeoutSeconds → cfg.Daemon.InvocationTimeoutSeconds
