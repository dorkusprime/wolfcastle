# Spec: Rendering contract for RenderContext methods

Write a spec documenting the rendering contract formed by the three RenderContext methods (Task, NodeState, AuditState). The spec must cover: output format and section ordering for each method, edge-case behavior (empty fields omitted, breadcrumb cap of 10, reference inlining size threshold of 8000 bytes), division of responsibility between domain-side rendering and ContextBuilder composition, and how the three outputs are assembled into a final iteration context. Addresses audit gap gap-render-context-methods-3.
