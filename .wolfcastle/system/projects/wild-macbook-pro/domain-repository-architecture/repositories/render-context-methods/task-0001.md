# Implement AuditState.RenderContext method

Add a RenderContext() string method to AuditState in internal/state/render.go. This method extracts the audit-related rendering from pipeline/context.go lines 163-179 into the domain type itself.

The method should render:
- Recent breadcrumbs (last 10) under a '## Recent Breadcrumbs' header, formatted as '- [timestamp] task: text'
- Audit scope description under a '## Audit Scope' header when scope is non-nil

Return empty string when there are no breadcrumbs and no scope (do not emit empty sections).

The current code in buildIterationContext that handles this lives at pipeline/context.go:163-179. The render method moves this logic to the domain type so that ContextBuilder can simply call ns.Audit.RenderContext() instead of inlining the rendering.

Do not modify callers yet. The ContextBuilder migration happens in a later task (internal/pipeline/ContextBuilder). This task only adds the new method and its tests.
