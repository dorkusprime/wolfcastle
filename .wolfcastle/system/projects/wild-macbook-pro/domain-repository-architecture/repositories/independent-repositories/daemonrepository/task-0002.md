# Test DaemonRepository with testutil.Environment

Create internal/daemon/repository_test.go with tests using testutil.NewEnvironment(t):

1. TestDaemonRepository_PIDLifecycle: Write PID, read it back, verify match, remove, verify read returns error.
2. TestDaemonRepository_StopFile: Verify HasStopFile is false initially, write stop file, verify HasStopFile is true, remove, verify false again.
3. TestDaemonRepository_LogDir: Verify LogDir returns the expected path (systemDir/logs).
4. TestDaemonRepository_ReadPID_Missing: Verify ReadPID returns an error when no PID file exists.
5. TestDaemonRepository_RemovePID_Idempotent: Verify RemovePID does not error when file is already gone.
6. TestDaemonRepository_RemoveStopFile_Idempotent: Same for stop file.

Use env.Root as the wolfcastleRoot argument to NewDaemonRepository.
