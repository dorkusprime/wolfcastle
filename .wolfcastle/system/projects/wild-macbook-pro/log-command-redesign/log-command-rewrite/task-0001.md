# Rewrite wolfcastle log cobra command

Replace follow.go with a new implementation. Register flags: --follow/-f (bool), --thoughts/-t (bool), --interleaved/-i (bool), --json (bool), --session/-s (int, default 0). Remove --lines and --level flags. The command should: (1) resolve the session from --session flag using logrender.SessionFiles(), (2) detect whether daemon is running to set implicit --follow, (3) select the output mode (summary/thoughts/interleaved/json), (4) delegate to the appropriate logrender renderer. Keep the 'follow' alias for backward compat but update the help text.
