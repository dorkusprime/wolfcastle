# Handle mutual exclusivity and last-wins semantics

When multiple output mode flags are passed (--thoughts, --interleaved, --json), the last one on the command line wins. Cobra doesn't natively support last-wins for bool flags, so implement this by checking raw os.Args order or by using a custom flag type. Write tests covering: single flag, multiple flags (last wins), no flags (defaults to summary), and --json output (raw NDJSON passthrough with no formatting).
