# Wire interleaved renderer into execute command

When wolfcastle execute runs in non-daemon mode, start a goroutine that tails the NDJSON log file being written by the execute loop and renders interleaved output to stdout using logrender.InterleavedRenderer. The goroutine should start after the logger creates its first iteration file and stop when execution completes. Use a done channel or context cancellation for clean shutdown. The rendering goroutine reads from the log file (not from memory), so the display code path is identical to wolfcastle log --interleaved --follow.
