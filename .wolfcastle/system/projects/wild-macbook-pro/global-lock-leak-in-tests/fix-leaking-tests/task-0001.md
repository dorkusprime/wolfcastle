# Redirect GlobalLockDir in start_test.go

Three test functions in cmd/daemon/start_test.go invoke the start command without redirecting GlobalLockDir, causing them to acquire the real ~/.wolfcastle/daemon.lock:

- TestStartCmd_ValidationErrors (line ~18)
- TestStartCmd_ValidationWarnings (line ~41)
- TestStartCmd_VerboseFlag (line ~120)

Add the standard redirection pattern to each function, immediately after creating the test environment:

    lockDir := t.TempDir()
    dmn.GlobalLockDir = lockDir
    defer func() { dmn.GlobalLockDir = "" }()

The established pattern is visible in start_validation_test.go for reference. Ensure the dmn import (internal/daemon) is present.
