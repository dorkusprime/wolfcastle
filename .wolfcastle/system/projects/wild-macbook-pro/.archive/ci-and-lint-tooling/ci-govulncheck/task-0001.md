# Add govulncheck job to CI workflow

Add a new job to .github/workflows/ci.yml that runs govulncheck. Use 'go install golang.org/x/vuln/cmd/govulncheck@latest' then 'govulncheck ./...' (or use the golang/govulncheck-action if preferred). Run on ubuntu-latest with go-version: stable. This should be its own job parallel to the existing lint, build-and-test, cross-compile, smoke-tests, and integration-tests jobs.
