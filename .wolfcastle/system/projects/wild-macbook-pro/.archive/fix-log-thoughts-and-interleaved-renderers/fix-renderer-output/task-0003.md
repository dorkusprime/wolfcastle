# Fix interleaved renderer to match spec

After diagnosing the root cause, fix the interleaved renderer so that wolfcastle log --interleaved produces spec-compliant output: stage headers with wall-clock timestamps and glyphs, agent text indented below. The spec example:

18:01:34 ▶ [execute] donut-stand-website/site-specification/task-0001
18:01:35     I'll start by creating the site specification document...
18:01:36     Reading the project requirements from the inbox item...
18:02:56 ✓ [execute] donut-stand-website/site-specification/task-0001 (1m22s)

Key formatting rules:
- Wall-clock timestamps in HH:MM:SS local time
- ▶ for stage starts, ✓ for success, ✗ for failure
- Agent thoughts indented 5 spaces after timestamp
- audit_report_written paths indented below audit completion line
- No sequence numbers, no log levels, no terminal markers

Both replay and follow modes must work. Update interleaved_test.go if the fix requires test changes.
