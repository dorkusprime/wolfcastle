# Implement ClassRepository

Implement ClassRepository in internal/pipeline/class_repository.go. The struct wraps PromptRepository and a goroutine-safe class map.

Constructor: NewClassRepository(prompts *PromptRepository) *ClassRepository

Methods:
- Reload(classes map[string]ClassDef): acquires write lock, replaces the internal class map. Called once at daemon startup with config.TaskClasses.
- Resolve(key string) (string, error): acquires read lock, looks up the class key, applies hierarchical fallback (split on '/' and try most-specific first, falling back to parent), then resolves the prompt file via PromptRepository.ResolveRaw('prompts/classes', matched+'.md'). Returns error if class key not found in map or prompt file missing.
- List() []string: acquires read lock, returns sorted keys from class map.
- Validate() []string: acquires read lock, iterates all configured classes, checks each has a resolvable prompt file, returns list of missing class keys.

ClassDef is defined in config; check existing type in internal/config/config.go. If it doesn't exist yet, define it there with whatever fields the task-classes spec requires.
