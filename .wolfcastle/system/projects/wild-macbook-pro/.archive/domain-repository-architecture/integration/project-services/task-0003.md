# Implement MigrationService

Implement MigrationService in internal/project/migration_service.go. Extract logic from the existing migrateToSystemLayout() and migrateOldConfig() functions in scaffold.go.

Constructor: NewMigrationService(config *config.ConfigRepository, root string)

MigrateDirectoryLayout(): check if system/ exists first (idempotent). Move directories and loose daemon files. Use os.Rename for moves. Create system/ if needed before moving.

MigrateOldConfig(): move root config.json to system/custom/config.json only if target absent. For config.local.json, read it, merge into existing local config, write via ConfigRepository.WriteLocal(). Remove source files after successful migration.

Both methods should be safe to call multiple times and on fresh installations (where there's nothing to migrate).
