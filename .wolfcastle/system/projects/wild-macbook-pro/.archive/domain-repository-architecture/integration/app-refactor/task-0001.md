# Refactor App struct fields

Refactor the App struct in cmd/cmdutil/app.go to replace raw config and Resolver with repository references.

Remove fields:
- WolfcastleDir string
- Cfg *config.Config
- Resolver *tree.Resolver

Add fields:
- Config *config.ConfigRepository
- Identity *config.Identity (nil if not configured)
- Prompts *pipeline.PromptRepository
- Classes *pipeline.ClassRepository
- Daemon *daemon.DaemonRepository
- Git git.Provider

Keep unchanged: Store, Clock, Invoker, JSONOutput (rename to JSON), Version

Update LoadConfig() to become Init():
1. Find .wolfcastle dir (existing logic)
2. Create all repositories: ConfigRepository, PromptRepository, DaemonRepository, git.Service, ClassRepository
3. Load config via ConfigRepository.Load()
4. Extract Identity from config (may be nil if not configured)
5. If identity present: create StateStore and call Classes.Reload(cfg.TaskClasses)
6. If identity absent: State and Identity remain nil

Add RequireIdentity() error method that returns an error if Identity is nil, replacing RequireResolver().
