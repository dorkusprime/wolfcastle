# Test daemon auto-archive timer

Write tests for the auto-archive daemon integration. Cover: (1) nodes completed within threshold are NOT archived, (2) nodes completed past threshold ARE archived, (3) auto-archive disabled in config skips all archiving, (4) only top-level orchestrators are eligible (not nested nodes), (5) already-archived nodes are not re-processed, (6) archive errors are logged but don't crash the daemon, (7) the timer respects the configured interval. Use the daemon's existing test patterns (clock.Mock, StateStore with temp dirs). Target >90% coverage.
