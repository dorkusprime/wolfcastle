# Add archive delete CLI command

Create cmd/archive_delete.go with a cobra command registered under archiveCmd. Usage: wolfcastle archive delete --node <addr> --confirm. The --confirm flag is required; if absent, print a warning and exit with error (do not prompt interactively). The command loads the RootIndex, validates the node is archived, calls deleteArchivedNode, and reports success. Support --json output. Register in init() under archiveCmd.
