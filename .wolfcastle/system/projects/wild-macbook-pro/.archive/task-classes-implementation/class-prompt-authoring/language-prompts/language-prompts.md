# Language Prompts

Author behavioral prompts for all 20 language classes. Each file goes under internal/project/templates/prompts/classes/coding/ (e.g., coding/go.md, coding/python.md).

Languages: python, javascript, typescript, java, csharp, go, rust, cpp, c, ruby, php, swift, kotlin, scala, shell, sql, r, lua, elixir, haskell, dart.

Each prompt is 40-80 lines covering: idiomatic style, specific build/test tool invocations, testing conventions, and common pitfalls. These REPLACE coding/default.md for classified tasks (not additive). Must be researched: current ecosystem tools, recent language changes, modern paradigms. Use 'prefer X' not 'always use X'. ZERO wolfcastle content.
