# Implement MigrateStagesFormat()

Add a new method MigrateStagesFormat() error to MigrationService in internal/project/migration_service.go. Algorithm: (1) For each tier file that exists (base/config.json, class/config.json, local/config.json), read the raw JSON. (2) Check if pipeline.stages is an array; if it is a map or absent, skip (idempotent). (3) If it is an array, convert: extract the 'name' field from each element to build stage_order []string, then build a map[string]any keyed by name with the name field removed from each value. (4) Write the converted JSON back to the tier file. (5) Handle edge cases: stages array is empty (produce empty map and empty stage_order), stages with missing name fields (skip with warning logged), duplicate names in array (last wins). Must be called after MigrateOldConfig() in the ScaffoldService.Reinit() call chain.
