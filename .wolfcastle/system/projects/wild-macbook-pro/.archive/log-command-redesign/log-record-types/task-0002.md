# Verify existing record types have required fields

Audit every call site that emits stage_start, stage_complete, planning_start, planning_complete, and audit_report_written records. The log command spec requires these fields: stage_start needs stage type and node address; stage_complete needs stage type, exit code, and timestamp (for duration calculation); audit_report_written needs the report file path; planning_start/planning_complete need node address. Grep for each record type in the daemon package, verify the fields, and add any that are missing. Write tests confirming the record shapes.
