## Summary

SummaryRenderer implements both replay and follow modes for the wolfcastle log summary view. Replay buffers all stage completion records, computes column widths, and writes aligned output with ✓/✗ glyphs and human-readable durations. Follow streams stage_start lines immediately with ▶ glyphs, then prints completion lines with duration as each stage finishes. Both modes handle planning records, audit report paths, daemon stage filtering, and edge cases (missing starts, nil exit codes, empty channels). The implementation is clean, well-tested (22 tests, race-clean), and faithful to the log-command-design spec.
