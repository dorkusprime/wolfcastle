# Add follow mode to summary renderer

Extend the SummaryRenderer with a Follow(ctx context.Context, records <-chan Record) method. In follow mode, stage_start records print immediately with the ▶ glyph: '▶ [stage-label] node-address'. When the matching stage_complete arrives, print the completion line with ✓ or ✗ glyph and duration: '✓ [stage-label] node-address (duration)'. The renderer must buffer the start timestamp keyed by (stage, node, task) to compute duration when the completion record arrives. audit_report_written records print indented below the most recent audit completion. The method returns when the context is cancelled or the records channel closes.
