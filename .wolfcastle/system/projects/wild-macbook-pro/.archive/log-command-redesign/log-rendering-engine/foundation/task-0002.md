# Implement compact duration formatting

Create internal/logrender/duration.go. Write a FormatDuration(d time.Duration) string function that produces compact human shorthand with no spaces: 34s, 1m22s, 12m5s, 1h3m. Rules: durations under 1 minute show seconds only (34s). Durations under 1 hour show minutes and seconds (1m22s), but drop the seconds component if it's zero (2m). Durations 1 hour or more show hours and minutes (1h3m), dropping zero minutes (1h). Never show sub-second precision. The function computes duration from the difference between two record timestamps; it takes a single time.Duration as input.
