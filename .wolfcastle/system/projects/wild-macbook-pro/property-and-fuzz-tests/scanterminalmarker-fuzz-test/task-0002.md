# Implement extractAssistantText fuzz test

Implement a companion fuzz test (func FuzzExtractAssistantText) for the extractAssistantText helper in internal/daemon/iteration.go.

This function parses arbitrary JSON and is called on every line of model output. It must handle malformed JSON gracefully.

Key implementation details:
- Use Go's native fuzzing (testing.F with f.Fuzz).
- Single string argument, call extractAssistantText(input), assert no panic.
- Seed corpus with:
  * Valid assistant envelopes in both simple and Claude Code formats
  * Valid result envelopes in both text and result field formats
  * Deeply nested JSON objects
  * Very large JSON payloads
  * JSON with unexpected types (numbers where strings expected, arrays where objects expected)
  * Non-JSON input that starts with '{'
  * Truncated JSON at various points
