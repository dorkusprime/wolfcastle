# Update renderers to prefer pre-computed duration

In internal/logrender/summary.go and internal/logrender/interleaved.go, modify the stage_complete and planning_complete handlers: if the record DurationMS is non-nil, convert it to time.Duration and use that instead of computing r.Timestamp.Sub(startTime). If DurationMS is nil, fall back to the existing timestamp-diff logic. This ensures backward compatibility with older log files that lack the field. Apply the same pattern in both Replay and Follow code paths.
