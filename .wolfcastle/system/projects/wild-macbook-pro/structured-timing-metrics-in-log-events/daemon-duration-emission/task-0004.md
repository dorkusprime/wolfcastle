# Add unit tests for duration_ms emission

Write or extend tests to verify that stage_complete and planning_complete log records contain a duration_ms field with a non-negative integer value. The tests should verify: (1) duration_ms is present in stage_complete records from iteration.go, (2) duration_ms is present in stage_complete records from stages.go, (3) duration_ms is present in planning_complete records. Use the existing test patterns in the daemon package. If a test logger or mock logger captures log records, assert the field exists and is >= 0.
