# Add duration_ms to planning_complete records

In internal/daemon/planning.go, record time.Now() immediately before the planning_start log call (~line 199). After invocation, compute elapsed := time.Since(startTime) and add "duration_ms": elapsed.Milliseconds() to the planning_complete log map (~line 214). The field name is duration_ms (not planning_duration_ms) for consistency across all completion record types.
