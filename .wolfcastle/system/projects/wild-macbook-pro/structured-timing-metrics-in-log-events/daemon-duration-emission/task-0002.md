# Add duration_ms to intake stage_complete records

In internal/daemon/stages.go, record time.Now() immediately before each intake invocation's stage_start log call (~line 179). After invocation, compute elapsed := time.Since(startTime) and add "duration_ms": elapsed.Milliseconds() to the stage_complete log map (~line 233). The timing should be per-item (each inbox item gets its own duration), not total batch time.
