# Add CommitPrefix field to GitConfig and set default

Add a CommitPrefix string field to the GitConfig struct in internal/config/types.go with JSON tag commit_prefix. In Defaults() in internal/config/config.go, set CommitPrefix to "wolfcastle". The field should be a plain string: empty string means no prefix, any other value is used as-is. Update the validation in internal/config/validate.go if it checks GitConfig fields. Update existing tests in internal/config/ that construct GitConfig literals to include the new field so they continue to compile.
