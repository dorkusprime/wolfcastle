# Implement restoreNode function

Add a restoreNode method to the archive service layer (internal/daemon/archive.go or a shared archive service, depending on where CLI access is cleanest). The function reverses archiveNode: for each node in the subtree (depth-first via collectSubtree), move .archive/{addr}/ back to {addr}/. Then mutate the RootIndex: remove from ArchivedRoot, append to Root, clear Archived and ArchivedAt on all subtree entries. Preconditions: node must exist in RootIndex.Nodes with Archived==true, must be in ArchivedRoot. Return clear errors for precondition violations. Reference the spec's Section 3 Restore for exact contract.
