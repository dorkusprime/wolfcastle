# Implement deleteArchivedNode function

Add a deleteArchivedNode method to the archive service layer. Preconditions: node must exist in RootIndex.Nodes with Archived==true, must be in ArchivedRoot. Procedure: (1) Remove .archive/{addr}/ and all descendant directories via os.RemoveAll. (2) Mutate RootIndex: remove all subtree entries from Nodes map, remove root address from ArchivedRoot. (3) Write updated index. Markdown rollup files in .wolfcastle/archive/ are NOT deleted (they are permanent records per spec Section 3). Return clear errors for precondition violations.
