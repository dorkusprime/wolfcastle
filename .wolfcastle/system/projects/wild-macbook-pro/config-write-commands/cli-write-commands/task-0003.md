# Implement config append command

Create cmd/config/append.go with the 'config append' cobra command.

Usage: wolfcastle config append <key> <value> [--tier local|custom]

Behavior:
- Takes exactly 2 positional args: key (dot-notation path) and value
- Parses value as JSON first; falls back to bare string
- Inside ApplyMutation: uses GetPath to read the current value at the path
  - If the value is an array ([]any), append the parsed value and SetPath the result
  - If the value is nil/missing, create a new single-element array and SetPath it
  - If the value exists but is not an array, return an error: '<key> is not an array'
- --tier flag defaults to 'local', accepts 'local' or 'custom' only
- Human output: 'Appended <value> to <key> in <tier>/config.json'
- JSON output: output.Ok envelope

Register in register.go.
