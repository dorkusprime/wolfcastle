# Implement config unset command

Create cmd/config/unset.go with the 'config unset' cobra command.

Usage: wolfcastle config unset <key> [--tier local|custom]

Behavior:
- Takes exactly 1 positional arg: key (dot-notation path)
- Uses app.Config.ApplyMutation to write nil at the path (DeletePath), which DeepMerge interprets as key deletion
- --tier flag defaults to 'local', accepts 'local' or 'custom' only
- Human output: 'Unset <key> from <tier>/config.json'
- JSON output: output.Ok envelope

Register in register.go.
