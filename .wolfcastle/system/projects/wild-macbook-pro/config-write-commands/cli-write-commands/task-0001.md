# Implement config set command

Create cmd/config/set.go with the 'config set' cobra command.

Usage: wolfcastle config set <key> <value> [--tier local|custom]

Behavior:
- Takes exactly 2 positional args: key (dot-notation path) and value
- Parses value as JSON first (handles numbers, booleans, null, objects, arrays); falls back to treating it as a bare string
- Uses app.Config.ApplyMutation(tier, func) to read-modify-write the tier overlay
- Inside the mutation function, uses SetPath to set the value at the parsed path
- --tier flag defaults to 'local', accepts 'local' or 'custom' only
- Human output: 'Set <key> = <value> in <tier>/config.json'
- JSON output: output.Ok envelope

Register the command in register.go via configCmd.AddCommand(newSetCmd(app)).
