# Implement config remove command

Create cmd/config/remove.go with the 'config remove' cobra command.

Usage: wolfcastle config remove <key> <value> [--tier local|custom]

Behavior:
- Takes exactly 2 positional args: key (dot-notation path) and value
- Parses value as JSON first; falls back to bare string
- Inside ApplyMutation: uses GetPath to read the current value at the path
  - If the value is an array, find and remove the first matching element, SetPath the result
  - If the value is not an array, return error: '<key> is not an array'
  - If the element is not found in the array, return error: '<value> not found in <key>'
  - Comparison should use JSON equality (marshal both to JSON and compare strings) for reliable matching of numbers, strings, objects
- --tier flag defaults to 'local', accepts 'local' or 'custom' only
- Human output: 'Removed <value> from <key> in <tier>/config.json'
- JSON output: output.Ok envelope

Register in register.go.
