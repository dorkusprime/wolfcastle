# Register knowledge command group and add shell completions

Create cmd/knowledge/register.go with a Register(app, parent) function that:

1. Creates the 'knowledge' parent command with a short description
2. Adds add, show, edit, and prune as subcommands
3. Assigns a GroupID (use 'docs' group alongside spec/adr commands, or create a new group if more appropriate)

Register the knowledge command group in cmd/root.go alongside other command registrations. Add shell completion support in the existing completions file for the new commands (node address completion for --node flags if applicable).
