# Implement wolfcastle knowledge add command

Create cmd/knowledge/ directory and implement the 'wolfcastle knowledge add "<entry>"' command. The command:

1. Takes a positional argument: the knowledge entry text
2. Calls app.RequireIdentity() to get the current namespace
3. Reads the knowledge config for max_tokens from app.Config
4. Uses internal/knowledge.CheckBudget to verify the entry won't exceed the token budget
5. If over budget, prints an error: 'Knowledge file exceeds budget (N/M tokens). Run wolfcastle knowledge prune to review and consolidate.' and exits with non-zero status. No silent truncation.
6. If within budget, calls internal/knowledge.Append to add the entry
7. Supports --json output via app.JSON

Follow the pattern established by cmd/spec.go and other command files.
