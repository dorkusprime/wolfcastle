# Author research.md and architecture.md

Create two non-language class prompts:

1. internal/project/templates/prompts/classes/research.md — Research/information gathering discipline. 30-60 lines. Covers: source citation and verification, accuracy over speed, structured output format, distinguishing primary from secondary sources, handling conflicting information, when to stop researching and start synthesizing. Research best practices from library science, competitive analysis frameworks, and systematic review methodology.

2. internal/project/templates/prompts/classes/architecture.md — System design discipline. 30-60 lines. Covers: dependency analysis, failure mode thinking, decomposition principles, tradeoff documentation, constraint identification, how to evaluate alternatives (not just list them), when to write ADRs. Research current architecture decision frameworks (C4 model, arc42, etc.).

ZERO wolfcastle-specific content. No references to wolfcastle commands, markers, or system mechanics.
