# Framework Prompts

Author behavioral prompts for all framework classes. Each file goes under a language subdirectory in internal/project/templates/prompts/classes/coding/ (e.g., coding/typescript/react.md, coding/python/django.md).

Frameworks: typescript/react, typescript/vue, typescript/angular, typescript/nextjs, typescript/svelte, javascript/react, javascript/node, python/django, python/fastapi, python/flask, ruby/rails, ruby/sinatra, java/spring, csharp/dotnet, php/laravel, php/symfony, kotlin/android, swift/ios, dart/flutter, elixir/phoenix, rust/actix, rust/tokio.

Each prompt is 30-60 lines. Framework prompts must NOT repeat language fundamentals (the fallback mechanism handles that). Cover framework-specific patterns, testing approaches, and common pitfalls. Researched and current. ZERO wolfcastle content.
