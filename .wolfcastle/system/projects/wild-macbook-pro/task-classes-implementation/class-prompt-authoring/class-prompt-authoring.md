# Class Prompt Authoring

Author behavioral prompt files for every class defined in the task-classes spec. Each prompt must be researched (not just training data), contain ZERO wolfcastle-specific content, and read like it was written by a senior practitioner.

Three categories of prompts:
1. Language classes (20 languages) under coding/ — 40-80 lines each, complete replacements for coding/default.md
2. Framework classes (20+ frameworks) under coding subdirectories — 30-60 lines each, do not repeat language fundamentals
3. Non-language classes (9 disciplines) directly under classes/ — 30-60 lines each, including writing.md which references writing/voice.md

All prompts must follow the authoring guidelines in the task-classes spec: prefer X over always use X, cover style/build/test/pitfalls, no wolfcastle internals.
