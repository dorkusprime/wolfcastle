Rewrite of docs/humans/configuration.md as the how-it-works guide. Covers three-tier directory structure with visual tree, merge semantics (deep-merge objects, replace arrays, null deletion) with before/after examples, common customizations (models, pipeline stages, class overrides, prompt overrides, rule fragments, audit scopes), wolfcastle config show with --section filtering, and CLI mutation commands (set/unset/append/remove) with examples. Links to Config Reference for field details and Task Classes for class customization.
