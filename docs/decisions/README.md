# Architecture Decision Records

This directory contains Wolfcastle's ADRs: records of significant architectural decisions made during design and development.

ADRs capture the context, decision, and consequences so future contributors (human or AI) understand *why* the system is built the way it is.

## Format

Each ADR follows the structure defined in [ADR-001](001-adr-format.md): Status, Date, Context, Decision, Consequences.

## Naming

Current files use sequential numbering (design phase, single author). Once Wolfcastle is self-hosting, ADRs will use ISO 8601 timestamp prefixes per [ADR-011](011-timestamp-filenames-for-docs.md): `2026-03-12T18-45Z-title.md`.
